# Music Web Player inspired from Spotify

A personal project built while learning modern web development. This app is inspired by Spotify’s interface and core features, but implemented from scratch as a way to practice frontend architecture, state management, and API integration.

# Features

- Play tracks and control playback with a custom audio player.  
- Switch between playlists, albums, or search results.  
- Responsive layout that works on both desktop and mobile.  
- Clean UI inspired by Spotify’s design language, but implemented in my own way.

# Tech stack

- Frontend: HTML, CSS, JavaScript (or React / Vue, if you used it)  
- Backend (if any): Node.js / Express / your stack  
- Music data: Spotify Web API / or your own mock data

# What I learned

- Managing state for an audio player and playlist navigation.  
- Handling asynchronous API calls and loading states.  
- Structuring components and keeping the UI code maintainable.

This project is a work‑in‑progress, so some features may still be incomplete or experimental.
